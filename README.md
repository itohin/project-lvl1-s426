# Brain-games
Set of mini games launched from the console

## Installation

composer global require itohin/brain-games

## Start IsEven Game

brain-even

## Start Calculator Game

brain-calc

## Start GCD Game

brain-gcd

## Start Progression Game

brain-progression

## Start IsPrime Game

brain-prime
