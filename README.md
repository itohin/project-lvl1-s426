# Brain-games
Set of mini games launched from the console

## Installation

composer global require itohin/brain-games
[![asciicast](https://asciinema.org/a/STf8ZIkxwDr2g0dmWAZ80dsFO.svg)](https://asciinema.org/a/STf8ZIkxwDr2g0dmWAZ80dsFO)

## Command to launch

brain-games

## Start IsEven Game

brain-even
